Shell アクション

シェルを通してプロセスの実行などを行なう

do_run(cmd, stdin=None, curdir=None, shell=True, error_on_failed=False)¶

コマンドの実行

- サンプルシナリオ

- action: shell
  cmd: ls -l /tmp

- action: debug
  output: <<last.out.stdout>>
パラメタ:
  • cmd (string) – 実行するコマンド
  • curdir (string) – 実行時のカレントディレクトリ。指定しない場合は zaffy 実行時のカレントディレクトリ
  • shell (bool) – シェルを介して実行する (default:True)
  • error_on_failed (bool) – コマンドの失敗時(return code != 0) にテストを error 扱いにする (default:False)
戻り値:

  • stdout (string) - 実行したコマンドの標準出力
  • stderr (string) - 実行したコマンドの標準エラー
  • returncode (int) - 実行したコマンドの終了ステータス
