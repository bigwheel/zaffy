local¶

class actions.local.Local(setting, params_obj)¶

Local アクション

シナリオ単位のローカル変数を保存する

上書き可能なローカル変数をシナリオ内で定義する。シナリオ内でのみ有効なため、シナリオ A が、シナリオ B (local アクションを実行する) を require しても、シナリオ A から B のローカル変数を参照することはできない。

注釈

シナリオ実行単位でグローバルに参照したい場合は const を使用する。ただし、定数なので上書きは不可能。(zaffy にグローバルな変数は存在しない)
