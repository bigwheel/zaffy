debug¶

class actions.debug.Debug(setting, params_obj)¶

Debug アクション

アクションの実行結果などデバッグ用の出力を行なう。 事前に実行した別アクションの実行結果や関数実行結果の出力に用いる。

注釈

通常の用途ではアクションの共通パラメータである _debug を使うことを推奨

- サンプルシナリオ

- action: http.get
  url: http://yahoo.co.jp/

# 直前のアクションの実行結果から HTTP STATUS を取得して表示
- action: debug
  result: <<last.out.status>> #=> 200 と表示
do_debug(global_env, **params)¶

do_rawprint の省略呼び出し

do_print(global_env, **params)¶

与えられた文字列値を python の基本データ型のリテラルとして解釈した後に print する

- サンプルシナリオ

- action: debug.print
  v1: "1 + 2 + 3" #=> 6 と表示
  v2: "[1, 2] + [3, 4]" #=> [1, 2, 3, 4] と表示
do_rawprint(global_env, **params)¶

与えられた値を文字列化して表示する

- サンプルシナリオ

- action: debug  # 省略時は rawprint の呼び出し
  v1: "1 + 2 + 3" #=> 1 + 2 + 3 と表示
