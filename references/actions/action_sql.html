do_select(driver, sql, db, host=None, user=None, password=None, port=3306, charset='utf8')¶

参照系のクエリ(SELECT, SHOW など)を実行し、タプル形式で結果を取得する

- サンプルシナリオ

- action: sql.select
  driver: mysql
  host: localhost
  db: user_db
  user: root
  password: hogehoge
  sql: select * from user where user_id=1
  assert:
   - >
    out.rows[0] is eq
      (1, "nanoha", "2012-04-10 15:57:26"|todatetime, "2012-04-10 15:57:26"|todatetime)
パラメタ:
  • driver (string) – データベースのドライバ (mysql または sqlite のみ指定可能)
  • sql (string) – 実行するSQL文
  • db (string) – データベース名
  • host (string) – データベースホスト名 (driver が mysql の場合のみ必要)
  • user (string) – DB接続のためのユーザ (driver が mysql の場合のみ必要)
  • password (string) – DB接続のためのパスワード (driver が mysql の場合のみ必要)
  • port (int) – DB接続先のポート番号 (driver が mysql の場合のみ必要)
  • charset (string) – DB接続時の文字コード (driver が mysql の場合のみ必要)(mysql のみ必要)
戻り値:

  • rowcount (int) - 取得した行数
  • rows (list) - 取得した行(tuple)のリスト
do_selectdict(driver, sql, db, host=None, user=None, password=None, port=3306, charset='utf8')¶

参照系のクエリ(SELECT, SHOW など)を実行し、辞書形式で結果を取得する

- サンプルシナリオ

- action: sql.selectdict
  driver: mysql
  host: localhost
  db: user_db
  user: root
  password: hogehoge
  sql: select * from user where user_id=1
  assert:
   - >
    out.rows[0] is eq
      {
       'user_id': 1,
       'user_name': "nanoha",
       'created_date': "2012-04-10 15:57:26"|todatetime,
       'updated_date': "2012-04-10 15:57:26"|todatetime
      }
パラメタ:
  • driver (string) – データベースのドライバ (mysql または sqlite のみ指定可能)
  • sql (string) – 実行するSQL文
  • db (string) – データベース名
  • host (string) – データベースホスト名 (driver が mysql の場合のみ必要)
  • user (string) – DB接続のためのユーザ (driver が mysql の場合のみ必要)
  • password (string) – DB接続のためのパスワード (driver が mysql の場合のみ必要)
  • port (int) – DB接続先のポート番号 (driver が mysql の場合のみ必要)
  • charset (string) – DB接続時の文字コード (driver が mysql の場合のみ必要)
戻り値:

  • rowcount (int) - 取得した行数
  • rows (list) - 取得した行(dict)のリスト、行のキーはカラム名に対応する
do_update(driver, sql, db, host=None, user=None, password=None, port=3306, charset='utf8')¶

更新系のクエリ(UPDATE, INSERT など)を実行する

- サンプルシナリオ

- action: sql.update
  driver: mysql
  host: localhost
  db: user_db
  user: root
  password: hogehoge
  sql: update user set user_name="fate" where user_id=1
  assert:
   - out.rowcount is eq 1
パラメタ:
  • driver (string) – データベースのドライバ (mysql または sqlite のみ指定可能)
  • sql (string|list) – 実行するSQL文。リストを渡した場合は複数実行する
  • db (string) – データベース名
  • host (string) – データベースホスト名 (driver が mysql の場合のみ必要)
  • user (string) – DB接続のためのユーザ (driver が mysql の場合のみ必要)
  • password (string) – DB接続のためのパスワード (driver が mysql の場合のみ必要)
  • port (int) – DB接続先のポート番号 (driver が mysql の場合のみ必要)
  • charset (string) – DB接続時の文字コード (driver が mysql の場合のみ必要)
戻り値:

  • rowcount (int) - 更新した行数、sql に複数のSQL文を渡した場合は最後に実行した結果
  • rows (list) - 空リスト
