env¶

class actions.env.Env(setting, params_obj)¶

Env アクション

システムの環境変数を取得する。存在しない環境変数名を指定した場合は Undefined を返す

- サンプルシナリオ

- action: debug
  e1: <<env.PATH>> #=> 環境変数の Path を取得
  e2: <<env.HOGE>> #=> 存在しないので Undefined を取得
  e3: <<env.HOGE|d('xyz')>> #=> デフォルト値 'xyz' を取得
