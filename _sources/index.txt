Welcome to zaffy's documentation!
=================================

zaffy は pure python 製の振る舞いテストツールです。もし、あなたが Web API 等を開発していて、その振る舞いを保証したい場合にこのツールが役に立つはずです。類似のテストツールは様々ありますが、zaffy は特にテストを書くためのルールを短時間で習得できることを目指しています。
